# vim_setup
## plugin setup
###
```
set number

colo monokai

set tabstop=4
 
set shiftwidth=4
set softtabstop=4
set expandtab
 
filetype plugin indent on
 
set autoindent
 
set laststatus=2

let g:airline#extensions#tabline#enabled = 1

let g:airline_theme='luna'
```
